### 远程资源

对于稳定不变并且行数多的资源放在github上，

```
surge
 - [General] 通用设置
 - [Replica] 
 - [Proxy]   代理节点
 - [Proxy Group]     策略组
 - [Rule]   规则分流
 - [Host]
 - [MITM]   解析hostname
 - [Script] 放置脚本
 - [Panel]  面板
``` 

### 本地资源

对于经常改变的资源放到本地，例如：获取cookie
以及 节点订阅

private github仓库不能长久进行访问
